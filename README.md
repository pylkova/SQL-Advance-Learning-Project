# SQL Advance Learning Project

Практический учебный проект для освоения SQL и Git  
с фокусом на аналитику данных и data engineering.

---

## 🎯 Цель проекта
- системно изучать SQL Advance
- фиксировать прогресс через Git
- формировать портфолио с понятной структурой
- выработать рабочие привычки (коммиты, документация)
